Banco de Dados

A matéria de banco de dados tem como atribuição ensinar aos alunos a modelar banco de dados. Estimular a organização e soluções dos problemas, promovendo ações que esteja considerando respeito as normas estabelecidas. Habilidades cuja os alunos devem adquirir: Normalizar tabelas de banco de dados, levantar as necessidades de informações do sistema e estabelecer relações entre tabelas
